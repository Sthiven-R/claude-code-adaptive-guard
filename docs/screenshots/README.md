# Screenshots

The main `README.md` references three screenshots from this directory.
They are not yet checked in; capture them with the dev server running
(`npm run tauri dev` from `dashboard/`) and save the PNGs here.

| File | What to capture | Approx. size |
|---|---|---|
| `dashboard.png` | Full main window with stats header, both histograms, and at least one expanded decision card (showing breakdown). | 1280×900 |
| `welcome.png` | First-run welcome screen — temporarily uninstall the hook from the Settings modal so the welcome appears. Restore afterwards. | 1100×720 |
| `settings.png` | Settings modal open, showing hook status, Language toggle, and Paths section. | 1100×720 |

Naming and dimensions are not load-bearing; GitHub re-scales for the
README. The names just have to match the references in `README.md`.

Optional — record a short GIF of the live watcher (a Claude Code prompt
landing in the dashboard within a second) and save as `live-watching.gif`.
That is a future addition, not part of the v0.1 README.
